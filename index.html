What:

Theatre Lighting Controller

A custom-built lighting controller for theatre lighting control. The project included a micro-controller system, a custom PCB assembly, 240V wiring, test-and-tag certification, and a sturdy codebase.

Warman Design and Build Challenge

My team's Warman Design Challenge entry: a custom aluminium frame robot with a robust control system and subcontroller architecture, designed by mechanical, software, electrical, and mechatronics students.

DIANA Set Electrics

An unconventional engineering project for sure - custom lighting solutions for the DIANA: The Musical set. Custom addressable LED strip and bulbs, open-source in-house designed controllers on a redundant network infrastructure.

Booster K1 Power Supply

A custom power supply for the Booster K1 robot. This project included the PCB "interposer" module, and the enclosure power supply.

Hourglass

Another interesting and unconventional project for entertainment technology. A fully wireless Hourglass prop which decodes DMX over WiFi and outputs to an addressible LED array. This project was picked up by Core Electronics for an article on Wireless DMX and prop building.

OpenFixtures

OpenFixtures: my crazy attempt to make an open-source lighting fixture platform. Another interesting entertainment engineering project.

J64

A fun little project I did during my work at Jaycar Electronics. A cool custom game console - Raspberry Pi based RetroPie gaming system. 3D printed enclosure, and lots of fun!

NUsense

NUsense is the sub-controller for the NUgus soccer robot; the current platform for the University of Newcastle's RoboCup team. While the PCB is not my design, I have been responsible for maintenance and design updates to it.
